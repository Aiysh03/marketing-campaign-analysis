# Marketing Campaign Performance Analysis

## Project Overview
Analyzed marketing campaign performance across 5 digital channels and 10 campaigns using MySQL and Power BI. The project focuses on campaign efficiency, ROI analysis, customer engagement, and channel performance.

## Tools & Technologies
- MySQL
- Power BI
- DAX
- Excel

## Key KPIs
- CTR
- Conversion Rate
- ROI
- Revenue per Click
- Cost per Conversion

## Dashboard Pages
1. Executive Overview
2. Channel Performance Analysis
3. Campaign Insights

## Key Insights
- Weekend Promo and Referral Program generated the highest revenue.
- Email Loyalty and Black Friday campaigns showed the lowest performance.
- Google Ads and LinkedIn Ads delivered strong conversion efficiency.

## Conclusion

- The analysis identified key high-performing campaigns and marketing channels by evaluating KPIs such as CTR, Conversion Rate, Revenue per Click, and ROI. 
- Power BI dashboards and MySQL queries revealed that campaigns with higher marketing spend generally generated stronger revenue and conversions, while some campaigns underperformed despite high impressions. 
- The project demonstrates how SQL, statistics, and interactive dashboards can be used together to generate actionable business insights for marketing performance optimization.
